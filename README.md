# Retail Sales Analysis | Business Analytics Case Study
##  Project Overview
This project presents an end-to-end retail sales analysis completed using Microsoft Excel. The objective was to identify key sales trends, evaluate product category performance, and generate business recommendations to support strategic decision-making.
The analysis was conducted as part of a Business Analytics coursework using a retail case study involving mall sales and inventory management. The repository contains my own analysis, dashboards, visualizations, and recommendations.
