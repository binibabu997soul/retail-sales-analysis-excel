# Retail Sales Analysis | Business Analytics Case Study
###  Project Overview
This project presents an end-to-end retail sales analysis completed using Microsoft Excel. The objective was to identify key sales trends, evaluate product category performance, and generate business recommendations to support strategic decision-making.
The analysis was conducted as part of a Business Analytics coursework using a retail case study involving mall sales and inventory management. The repository contains my own analysis, dashboards, visualizations, and recommendations.
### Business Problem
A retail shopping mall sought to understand post-pandemic sales performance across product categories and store locations to improve profitability while maintaining customer satisfaction. The goal was to identify high-performing categories, optimize inventory allocation, and support data-driven business decisions.
### Objectives
- Analyze retail sales performance
- Identify top-performing product categories
- Perform Pareto (80/20) analysis
- Discover sales distribution across departments
- Generate business recommendations
- Build an interactive Excel dashboard
